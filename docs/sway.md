# Sway

Interface gráfica tilling window

Para iniciar, primeiramente importamos a configuração do diretório config, após executar o provisionamento.

`stow sway -t ~`

Isso irá copiar o arquivo de configuração do sway para o diretório `~/.config/sway`

Para que as configurações tenham efeito, é necessário reiniciar o ambiente gráfico com o comando `Super + Shift + c`



## Atalhos

O principio benefício do uso do Sway em meu workflow é o fato de ter o teclado como componente central, ou seja, posso abrir qualquer coisa via atalhos de teclado 


`Super + Enter`:  Abre o terminal.
`Super + b`: Abre o navegador
`Super + q`: Fecha a janela ativa
